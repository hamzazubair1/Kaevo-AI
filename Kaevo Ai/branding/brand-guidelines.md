# Branding

> Brand assets, guidelines, and design resources for Kaevo AI.

## Structure

```
branding/
├── logos/           # Logo files (SVG, PNG, ICO)
├── colors/          # Color palette files and swatches
├── fonts/           # Typography files
├── templates/       # Design templates (social, presentations)
└── brand-guidelines.md  # Quick-reference brand guide
```

## Usage

- See [Brand Guidelines](../docs/brand-guidelines.md) for complete brand standards
- All logos must be used according to the guidelines
- Colors and fonts must match the approved palette
- Templates are pre-configured with brand settings

## Adding Assets

1. Place files in the appropriate subdirectory
2. Use lowercase, kebab-case filenames
3. Include multiple formats where applicable (SVG + PNG)
4. Update this README if adding new categories
