# Portfolio

> Completed projects, case studies, and demos showcasing Kaevo AI's work.

## Structure

```
portfolio/
├── case-studies/    # Written case studies (Markdown)
├── demos/           # Demo projects and prototypes
└── README.md
```

## Case Study Template

```markdown
# [Project Name] — Case Study

## Client
Company name, industry, size

## Challenge
What problem did the client face?

## Solution
What did Kaevo AI build?

## Results
Quantifiable outcomes (time saved, cost reduced, revenue increased)

## Technologies Used
Tech stack for this project

## Testimonial
Client quote (with permission)

## Timeline
Project duration
```

## Adding a Case Study

1. Create a new Markdown file in `case-studies/`
2. Follow the template above
3. Include screenshots or recordings in `demos/`
4. Link from the website portfolio page
