# Agent B5: Outreach Email Agent

> Crafts personalized, compliant outreach sequences based on proposal insights.

---

## Identity

| Field | Value |
|-------|-------|
| **Name** | Outreach Email Agent |
| **ID** | `agent-b5-outreach-email` |
| **Version** | 1.0.0 |
| **System** | System 2 — AI Outbound Growth Engine |
| **Type** | Generative |
| **Model** | GPT-4o (primary), Claude 3.5 Sonnet (fallback) |
| **Trigger** | Proposal generated by Agent B4 |

---

## Purpose

To compose highly personalized, multi-touch email outreach sequences that leverage website analysis data to secure strategy meetings. Ensures all drafted communications comply with anti-spam regulations and company tone.

---

## Responsibilities

1. Ingest email variables (from B4) and business profile (from B1).
2. Select appropriate email sequence strategy based on opportunity score (B3).
3. Draft initial outreach email focusing on the primary identified gap.
4. Draft follow-up emails (value-add, case study, break-up).
5. Apply safety and compliance guardrails (CAN-SPAM/GDPR).
6. Queue the drafted sequence for human review.

---

## Processing Logic

```
Proposal Generated (B4)
       │
       ▼
Determine Strategy (based on Score from B3):
├── Score 80-100: Hot Sequence (3 emails, hyper-personalized)
└── Score 60-79: Strong Sequence (3 emails, industry-focused)
       │
       ▼
Draft Email 1 (Initial Outreach):
├── Subject: Relate to {primary_gap} or {business_name}
├── Body: Mention {primary_gap} observed on {website}
├── Value: Propose {solution} saving {estimated_savings}
└── CTA: Link to Proposal or Booking Calendar
       │
       ▼
Draft Email 2 (Follow-up):
├── Focus on Case Study / ROI
└── Soft CTA
       │
       ▼
Draft Email 3 (Break-up/Nurture):
├── Provide free resource
└── Leave door open
       │
       ▼
Run Compliance Check (Unsubscribe link, sender address)
       │
       ▼
Submit to Admin Dashboard Queue (Status: pending_human_review)
```

---

## Error Handling & Fallbacks

| Error Scenario | Response Strategy |
|----------------|-------------------|
| Missing contact email | Flag sequence as "needs_contact_info" for manual sales rep action |
| Compliance check fails | Re-generate or flag for critical manual review |
| Generation failure | Retry once, alert admin |
