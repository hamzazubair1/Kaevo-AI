# Kaevo AI — Development Roadmap (V2)

> Expanded 5-Phase enterprise development roadmap from MVP to Global AI SaaS.

---

## 🏢 Phase 1: MVP (AI Agency Platform - System 1)
**Focus:** Inbound lead capture, qualification, and delivery operations.
**Timeline:** Months 1–2

- **Website & Brand:** Launch high-converting Next.js landing site.
- **Agent A1 (Receptionist):** Deploy chat widget for instant visitor engagement.
- **Agent A2 (Qualification):** Implement structured lead scoring flow.
- **CRM Foundation:** Set up unified Supabase database for leads and deals.
- **Admin Dashboard (v1):** Basic pipeline management and lead tracking.
- **Service Delivery:** Launch 3 core services (Chatbots, Automation, Voice).

---

## ⚙️ Phase 2: Outbound AI Engine (System 2)
**Focus:** Proactive growth, automated discovery, and personalized outreach.
**Timeline:** Months 3–4

- **Agent B1 (Discovery):** Implement Google Maps/Search scraping pipelines.
- **Agent B2 (Analysis):** Deploy Playwright crawler for tech/gap detection.
- **Agent B3 (Scoring):** Activate multi-dimensional opportunity scoring.
- **Agent B4 & A3 (Proposals):** Unify proposal generation for both inbound and outbound.
- **Agent B5 (Email):** Implement drafting sequence with Admin Dashboard HITL approval queue.
- **Outbound Analytics:** Dashboard for tracking discovery volume, open rates, and conversion.

---

## 🤖 Phase 3: Internal AI Employees
**Focus:** Scaling operations without scaling headcount.
**Timeline:** Months 5–6

- **Project Management Agent:** Auto-updates task statuses based on git commits and emails.
- **Finance Agent:** Automates invoicing, payment tracking, and financial projections.
- **Content Agent:** Generates SEO-optimized blog posts and social media content based on recent case studies.
- **HR/Onboarding Agent:** Automates contractor/freelancer onboarding and training.

---

## 📊 Phase 4: Client AI Dashboards
**Focus:** Transparency, retention, and recurring revenue.
**Timeline:** Months 7–9

- **Client Portal:** Secure login for active clients.
- **Project Telemetry:** Real-time metrics on deployed automations (e.g., "Hours saved this week: 42").
- **Support Agent:** AI-first ticketing system for client requests.
- **Upsell Engine:** Automated recommendations for additional services based on client usage patterns.

---

## 🌍 Phase 5: AI SaaS Platform
**Focus:** Transitioning from agency services to a self-serve global platform.
**Timeline:** Months 10–18+

- **Self-Serve Agent Builder:** No-code visual interface for clients to configure their own agents.
- **Multi-Tenant Architecture:** Strict data isolation and scalable infrastructure.
- **Subscription Billing:** Stripe integration for usage-based and tiered subscriptions.
- **Agent Marketplace:** Allow third-party developers to publish and monetize agent templates.
- **Public API:** Enable enterprise clients to integrate Kaevo AI agents into custom workflows.

---
> _Note: Replaces Day 1 development roadmap. Retains all core objectives but maps them to the new System 1 / System 2 architecture._
